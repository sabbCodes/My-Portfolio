FoodOrdering App.

Restaurant App.

This is an application built for food vendors to make ordering and delivering of food seemingly easy and fast.

It's built using HTML, CSS and JavaScript. It contains all the foods available at the Restaurant, and visitors can add the various food items they wish to order by clicking on the "+" button beside each food.

For each food item being selected, the food is added to the "Your Order" list at the bottom of the App. The App also automatically sums the total prices of the items in the "Your Order" section and displays the amount.

There is a "remove" option for each item in the "Your Order" section, when clicked, the particular food item is removed and then "Total price" is updated.

When the "Complete Order" button is clicked, a payment page pops up and requires the user to input their card details to complete the order. A popup notification with the user's name shows when the payment has been verified.
