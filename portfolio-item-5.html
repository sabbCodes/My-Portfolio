TinderFor Dogs.

Tindog App.

A Tinder dating application, but for dogs.

It's built using HTML, CSS and JavaScript. It contains pictures and details of all dogs available for date, and can decide whether to like or reject a particular date using the 2 buttons at the bottom of the app.

A "LIKE" or "NOPE" stamp appears on each date, depending on which button the user clicks.

The app waits for some time before displaying the next date available.
