AnnoyingCookie Consent.

Modal Popup.

A cookie consent application built to annoy users by forcing them to input their details before using the site.

It's built using HTML, CSS and JavaScript. It shows the popup just when users begin to read the content of the page.

The close button does not work until the end of the whole process, and even though there is a decline button on the popup, users can not use it as it swaps position with the accept button each time the mouse hovers over it.

The app is especially funny as it displays a pirate laughing GIF at the end.
