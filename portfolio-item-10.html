PumpkinMeme Picker.

Pumpkin's Meme Picker.

An application that gives random cat memes based on user's current emotion.

It's built using HTML, CSS and JavaScript. User can to select their current emotion, and get cat memes matching the selected emotion.

The app only returns images in GIFs format if the "GIFs only" radio button is checked.

A click anywhere on the page closes the popup.
