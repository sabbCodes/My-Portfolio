Colorscheme Generator.

Color App.

An application built for designers and developers to help get color schemes for their work.

It's built using HTML, CSS, color API and JavaScript.

It lets users input the color they want through the color input field. User can select any mode from a range of different modes in the options field.

On clicking the "Get color scheme button", the application reach out to the color API and get 5 different colors for the scheme.

The app displays the different colors on the page as well as their respective hex codes.
